# Big Data Technology - Praktikum

## Deskripsi
Repositori ini berisi proyek praktikum mata kuliah **Big Data Technology** di Prodi Teknologi Informasi UIN Antasari.

## Teknologi
- **Python 3.10** – Bahasa pemrograman utama
- **PySpark** – Processing engine untuk pemrosesan data besar
- **MongoDB Atlas** – Cloud database (NoSQL)
- **Git & GitHub** – Version control dan kolaborasi

## Struktur Proyek
```
bigdata-project/
├── data/              # Penyimpanan data lokal
├── cloud_storage/     # Simulasi storage cloud
├── scripts/           # File Python (.py)
├── notebooks/         # Jupyter Notebooks (.ipynb)
├── reports/           # Output analisis
├── requirements.txt   # Daftar dependensi
└── README.md          # Dokumentasi proyek
```

## Instalasi
```bash
pip install -r requirements.txt
```

## Menjalankan Spark Job
```bash
python scripts/simple_job.py
```
